## Что умеет бот

- **Автоматическая рыбалка** — бот сам нажимает `F` для заброса удочки, ждёт появления мини-игры и начинает удерживать поплавок в зоне.
- **Цветовое сканирование полосы** — распознаёт цианового цвета прогресс-бар через захват экрана. Верхняя и нижняя половины полосы сканируются по разным цветовым профилям (`#20b5a1` и `#36e6bf`) для точного распознавания без ложных срабатываний на небо и фон.
- **Отслеживание жёлтого маркера** — находит позицию жёлтого индикатора внутри полосы и управляет им клавишами `A` / `D`.
- **Умное управление** — использует импульсный алгоритм: чем дальше маркер от центра, тем дольше удерживается клавиша. Мёртвая зона ±1.5px предотвращает дрожание.
- **GDI-оверлей** — рисует поверх игры прозрачный оверлей с визуализацией сканируемой зоны, позиций бара и маркера в реальном времени.
- **Watchdog** — если бот завис или потерял полосу дольше 60 секунд, автоматически кликает в нужную точку экрана и перезапускает цикл.
- **Таймер автовыключения** — можно задать время (часы / минуты / секунды), по истечении которого компьютер сам выключится.
- **Отключение звука игры** — кнопка для глушения аудио процесса игры без выключения системного звука.
- **Поддержка любых разрешений** — автоматически рассчитывает зону захвата и точку клика под любое разрешение экрана (480p, 720p, 1080p, 1440p, 4K, 8K и любое другое).

## Системные требования

- **OS:** Windows 10 / 11 (только Windows)
- **Python:** 3.14.4 или новее
- **Права:** администратор (бот запросит их автоматически при запуске)

---

## Установка

**1. Установи Python**  
Скачай с [python.org](https://www.python.org/downloads/) и при установке поставь галочку **Add Python to PATH**.

**2. Установи зависимости**  
Открой папку с ботом в терминале и выполни:
```
pip install -r requirements.txt
```

---

## Запуск

```
python NTE_FishingBot.py
```

При первом запуске Windows спросит разрешение администратора — нажми **Да**.

## Интерфейс

| Элемент | Описание |
|---|---|
| Кнопка **START** | Запустить бота |
| Кнопка **STOP** | Остановить бота |
| Поля **H / M / S** | Таймер автовыключения ПК |
| Кнопка ▶ таймера | Запустить таймер |
| Кнопка ✕ таймера | Отменить таймер |
| **Отключить оверлей** | Скрыть/показать GDI-оверлей поверх игры |
| **Отключить звук** | Заглушить аудио игры |
| **WD: Xs** | Оставшееся время до срабатывания watchdog |

---

## Горячие клавиши (игровые, управляются ботом)

| Клавиша | Действие |
|---|---|
| `F` | Заброс / подтверждение |
| `A` / `D` | Движение маркера влево / вправо |
| `ESC` | Выход из мини-игры при потере полосы |
